# recruitment

#### 介绍
基于Springboot+layui实现的招聘网站

#### 软件架构
前端使用layui完成；
后端分为求职者端，招聘者端和管理员端；
其中求职者端在Apply项目中；
招聘者端和管理员端在Company项目中；
